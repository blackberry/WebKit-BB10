This test can be used to verify that we preserve zoom with respect to the specified viewport width on rotation.

  1. Zoom in on the page.
  2. This test PASSED if the specified value stays relatively constant (*) as you rotate the device between portrait- and landscape- orientation and vice versa.
    • Mobile Safari: innerWidth
    • RIM device: scale
(*) Mobile Safari: The difference in innerWidth should not exceed 1. RIM device: The difference in scale should not exceed 0.01.
