Video embed within a object element

This is a test of a video player rendered by an <object> element whose fallback content includes an <embed> element and some text.

  • Press the play button to start the video
  • While the video is playing, verify that the time display slider is updating automatically, that the fullscreen button is displayed, and that you can press the fullscreen button.
  • Verify that you do not see fallback content, which includes text which says, "File cannot be played".
